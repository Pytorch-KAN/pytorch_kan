PyTorch KAN: Kolmogorov-Arnold Networks

A PyTorch implementation of Kolmogorov-Arnold Networks (KAN) - a flexible, interpretable neural network architecture based on the Kolmogorov-Arnold representation theorem.

Get Started Tutorials

Flexible Basis Functions

Choose from a wide range of orthogonal polynomial families (Chebyshev, Legendre, Hermite, Laguerre) and other basis function types to find the optimal representation for your data.

Efficient Implementation

Optimized PyTorch implementation with adaptive approximation capabilities, allowing for better performance with fewer parameters compared to traditional neural networks.

Interpretable Architecture

Based on the Kolmogorov-Arnold theorem, KANs provide more interpretable and theoretically grounded neural networks, offering insights into the underlying function approximation.
